# Mirror Repository for ParAnnoying Messenger

ParAnnoying Messenger aims to be a messenger which aims on security only even when loosing user experience.
It should be able to theoretically overcome goverment survaillance by using air gap mechanics.
Development started in 2018 as mainly a hobby and educational project and evolved several times.
There are several parts working and setup to be fully functional, still currently the development is discontinued while a lot more of works needs to be done.
The Code is mostly complete, still private and sensitiv information are stripped out forcefully.
As well as there are still big features which have to be finished or fixed.
- Airgap Adapter is working only in very few scenarios properly
- E2E Test is very flaky and Token system seems to be compromised
- many features arent tested completely
- many dependencies are outdated and might not work with newer android versions
